# EDR CSV Process Tree Viewer

ブラウザだけで動作する、オフライン前提の EDR / フォレンジック補助ツールです。CSV はローカルでのみ処理され、外部送信しません。

## 現在の動作方針

- **1つのプロセス作成イベント = 1ノード**
- **関連イベント** は `InitiatingProcessID` と `Initiating Process Creation Time` が、そのノードの `ProcessID` と `Process Creation Time` に一致するものを紐付け
- **親子関係** は、子ノードの `InitiatingProcessID` と `Initiating Process Creation Time` を使って親ノードを解決
- 親ノードがログ内に存在しない場合は、**推定親ノード** を補完表示

## 必須 / 推奨列

### 必須
- `ProcessID`
- `InitiatingProcessID`

### 強く推奨
- `Process Creation Time`
- `Initiating Process Creation Time`
- `ActionType` などのイベント種別列

### 任意
- `InitiatingProcessParentID`
- ホスト / デバイス識別列
- 表示に使いたい任意列

## 主な機能

- 数百MB級 CSV を想定したチャンク読み込み + Web Worker 解析
- 先頭プレビュー、ヘッダ有無指定、区切り文字 / 文字コード指定
- 表示項目の選択
- ルート選択、検索、表示フィルタ
- **一致ノード + 祖先 + 子孫のみ表示**（兄弟は非表示）
- 一致したノードの**強調表示**
- ノード選択時の関連イベント一覧表示
- 選択ノード直下への関連イベント四角ノード表示
- ノード詳細で**元ファイルの行番号情報**を表示
- 表示タイムゾーン変更
- **表示時刻フィルタ**（入力は表示タイムゾーン基準）
- 親未観測時の**推定親ノード補完**

## 表示フィルタ

複数キーワードを改行 / カンマ / セミコロン区切りで指定できます。

- `OR`: いずれか一致
- `AND`: すべて一致
- 一致したノードは色とバッジで強調
- 表示対象は **一致ノード + 祖先 + 子孫** のみ
- 兄弟ノードや無関係の枝は非表示

## 表示時刻フィルタ

- 解析済みデータに対して**再描画のみ**で適用
- 入力値は**表示タイムゾーン**の時刻として解釈
- 判定は、ノードに紐付いた create / related イベントの実時刻で行う

## 親推定

親ノードがログに無い場合でも、`InitiatingProcessID` と時系列情報から **推定親ノード** を補完します。

- 観測親が無い子ノードは、推定親の子としてぶら下がる
- 詳細ペインで「親推定補完」「推定コンテキスト」を確認可能

## 使い方

1. `index.html` をブラウザで開く
2. CSV を選択してプレビュー読込
3. 必須列 / 推奨列を指定
4. 必要に応じて表示項目、タイムゾーン、表示フィルタを調整
5. 解析開始
6. ルートを選んで表示
7. ノードや関連イベントをクリックして詳細を確認
